A library to simplify working with Java, which includes basic functions

### Install
+ **With Maven**
  ```
  <dependency>
  	<groupId>com.github.levkoposc</groupId>
  	<artifactId>jtools</artifactId>
  	<version>0.1</version>
  </dependency>
  ```
+ **With Gradle**
  ```
  dependencies {
  	compile 'com.github.levkoposc:jtools:0.1'
  }
  ```
